## 1. 游戏概述

**迷雾森林逃生** 是一款基于 HTML5 Canvas 的浏览器生存射击游戏。玩家扮演一名被困在森林中的猎人，需要通过探索、战斗、收集资源，最终找到逃生车逃离森林。游戏采用“天数”循环机制，每天难度递增，形成 roguelite 式的成长体验。

**核心设定**：玩家受雇进入森林打猎，4天后老板未按时接应。食物耗尽、怪物出没，玩家必须尽快打倒怪物、坐上逃生车逃离森林。

```
                       ▼
┌─────────────────────────────────────────────────────────────┐

│  - 进入迷宫收集2把钥匙                                          │

│  - 收集迷宫内的弹药和宝箱                                       │
└──────────────────────┬──────────────────────────────────────┘
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  阶段3: 载具逃生 (Driving)                                     │

                       ▼

```

## 4. 核心系统设计

### 4.1 武器系统（24种武器）

武器分为两类：
- **基础武器（9种，等级1-9）**：手枪、步枪、弩、霰弹枪、火焰喷射器、冰冻枪、激光枪、加农炮、风暴枪
- **生成武器（15种，等级10-24）**：通过算法生成，循环使用9个名称

**武器属性公式**：
```javascript
price = round(920 + (level-9)*(level-5)*34) * (1.85 + level*28)
damage = round(22 + (level-9)*4.5)
fireDelay = max(0.08, 0.2 - (level-9)*0.004)
bulletSpeed = 850 + level*12
bulletLife = 1.15 + level*0.012
```

**设计亮点**：
- 武器等级与弹药等级绑定（非武器ID）
- 颜色循环：红、绿、紫、黄、青
- 等级≥3的武器可破石

### 4.2 经济系统

| 项目 | 计算方式 |
|------|----------|
| 弹药价格 | `14 + level*9 + level²*1.35` |
| 弹药包大小 | 等级≤3: 12发，否则10发 |
| 医疗包治疗 | `10 + clamp(level,1,5)*14` |
| 击杀奖励 | 普通: `max(5, level*5)`，Boss: `max(120, level*35)` |

### 4.3 难度缩放系统

```javascript
// 每天怪物数量
forest: 4 + (day-1)*2
maze: 固定6个刷新点

// 危险等级
danger = 1 + (day-1)*0.55

// 怪物属性
hp = ceil(1 + level*1.8)
speed = rand(54,82) + danger*5

// Boss（每5天）
level = max(6, danger + day)
hp = ceil(45 + level*8)
```

### 4.4 存档系统

**存储内容**（极简设计）：
```javascript
{
  day: 1,
  points: 0,
  weaponId: "pistol",
  ownedWeapons: ["pistol"],
  ammo: { "1": 30 }
}
```

**设计哲学**：只存元进度，不存进行中的游戏状态。死亡或退出后从当前天重新开始。

## 5. 渲染系统分析

### 5.1 森林渲染
- **树木**：基于网格的确定性噪声生成，每格96px
- **装饰物**：帐篷、原木、骨头、箱子、路标、篝火（320px网格）
- **岩石**：可破坏的障碍物，带碰撞检测
- **灌木丛**：可隐藏，提供隐身机制

### 5.2 迷宫渲染
- **墙壁**：程序化砖块纹理，带裂缝细节
- **网格背景**：32px网格线
- **门**：收集2把钥匙后变绿

### 5.3 怪物渲染
- **类型**：small（紫色）、normal（红色渐变）、boss（黑色）、mazeBig（深红）、chaser（追击型）
- **视觉效果**：尖刺身体、发光眼睛、伤害指示条、移动拖尾
- **等级显示**：怪物身上方显示等级或"BOSS"

### 5.4 天气系统
6种天气效果：

| 天气 | 视觉效果 |
|------|----------|
| rain | 斜线雨滴 |
| snow | 白色飘落雪花 |
| ash | 灰色灰烬 |
| fog | 半透明雾团 |
| toxic | 绿色毒雾 |
| redmoon | 红色覆盖层 |
| embers | 橙色余烬 |

## 8. 设计亮点

### 8.1 程序化生成
- **全部美术资源通过代码生成**：无外部图片、音频
- **SVG 武器图标**：基于武器属性动态生成
- **森林场景**：12种场景，每天根据哈希算法选择

### 8.2 渐进式难度
- 天数驱动的指数增长
- 武器升级提供正反馈
- Boss每5天出现，提供高额奖励

### 8.3 多阶段逃生
- 森林→迷宫→森林→驾车，形成节奏变化
- 迷宫提供"安全区"（相对线性的挑战）
- 驾车阶段增加紧张感（追击怪物）

### 8.4 隐藏机制
- **灌木丛隐藏**：停止移动后隐身，怪物变为漫游
- **岩石破坏**：高等级武器可破坏岩石开辟路径

## 10. 总结

《迷雾森林逃生》是一款设计精良的浏览器生存射击游戏，其核心优势在于：

1. **零依赖**：纯原生技术实现，部署简单
2. **完整的游戏循环**：探索→战斗→收集→升级→逃生的闭环
3. **渐进式难度**：天数驱动的 roguelite 体验
4. **程序化美术**：无需外部资源，包体极小
5. **跨平台**：浏览器原生运行，支持桌面和移动端

游戏在保持简洁的同时，通过三阶段循环、24种武器、12种场景、6种天气等设计，提供了丰富的可玩性和重玩价值。
